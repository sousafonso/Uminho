### Redes Neurais Artificiais
**Objetivo:** Definir um conjunto de valores para os pesos das conexões que minimizem a função de custo.

**Forward propagation:** cálculo do output para o respetivo vetor de input e do erro (dado o output e o output desejado).
**Backpropagation:** dado o erro, este é propagado para trás, ajustando o peso das conexões de modo a minimizar o erro (idealmente minimo global da função, no entanto em maior parte dos casos, um minimo local já é útil para o problema em causa). Este processo é baseado no cálculo do gradiente do custo.
