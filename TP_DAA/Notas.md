Modelos exigem valores numéricos para treinar o modelo. Portanto, precisamos de converter variáveis categóricas em valores numéricos.
Sugiro as seguintes equivalencias:
Para a luminosidade:
- DARK = 0
- LOW_LIGHT = 1
- LIGHT = 2

Para a chuva
- sem chuva = 0
- chuva fraca = 1
- chuva moderada = 2
- chuva forte = 3
